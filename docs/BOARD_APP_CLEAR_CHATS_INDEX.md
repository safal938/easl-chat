## üìä Feature Overview

### What It Does

Allows the Board App to clear all chat conversations that were initiated through the board app, while preserving chats created directly in the main EASL application.

### Key Features

- ‚úÖ Selective deletion (only pure board app chats)
- ‚úÖ Safe preservation (mixed-source chats protected)
- ‚úÖ Secure (Firebase authentication required)
- ‚úÖ User-scoped (users can only delete their own chats)
- ‚úÖ Detailed results (statistics on deleted chats/messages)

### API Endpoint

```
POST https://easl-board.vercel.app/api/board-app/clear-chats
```

### Quick Example

```javascript
const auth = getAuth();
const idToken = await auth.currentUser.getIdToken();

const response = await fetch('https://easl-board.vercel.app/api/board-app/clear-chats', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${idToken}`,
    'Content-Type': 'application/json',
  },
});

const result = await response.json();
console.log(`Cleared ${result.deletedChats} chats`);
```
